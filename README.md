# Simple Shopping Cart Program (Python)

This is a beginner-friendly Python program that simulates a simple shopping cart
using command-line input.

## What the Program Does
- Allows the user to enter food items
- Stores item names and prices in lists
- Lets the user quit by entering `q`
- Displays the list of items purchased
- Calculates and displays the total cost

## Concepts Used
- Lists
- While loop
- Break statement
- User input
- Basic arithmetic operations

## Sample Output

Enter the price of the Apple: $ 30
What food do you like to buy (q to quit): Bread
Enter the price of the Bread: $ 40
What food do you like to buy (q to quit): q
YOUR LIST
Apple Bread
Your total is $70.0


## How to Run
1. Save the file as `shopping-cart.py`
2. Run the program using: python shopping-cart.py
